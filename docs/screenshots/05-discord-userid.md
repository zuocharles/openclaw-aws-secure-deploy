## Overview

Your Discord User ID is a unique number that identifies you. OpenClaw uses it to ensure only YOU can control the bot.

**Estimated time:** 2 minutes

---

## Step 5.1: Enable Developer Mode

1. Open Discord (web or app)
2. Click the gear icon (User Settings) next to your username
3. Scroll down and click "Advanced"
4. Toggle ON "Developer Mode"
5. Close settings

## Troubleshooting

### "I don't see 'Copy User ID' option"
- Make sure Developer Mode is enabled (Step 5.1)
- Try restarting Discord
- Make sure you're clicking on your own name, not someone else's

### "The ID is different on mobile vs desktop"
- Your User ID is the same everywhere
- If they look different, one might be a message ID instead
- Always use the "Copy User ID" option, don't try to type it manually
